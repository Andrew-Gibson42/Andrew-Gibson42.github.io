Counting Calculator: Factorials, Permutations, and Combinations


Factorials:

A factorial, symbolized by !, of an integer is the product of that number and all integers below it, above 0. For example, 3! = 3 * 2 * 1 = 6. For example, 3! is the number of ways 3 options could be picked, in order.

Permutations

A permutation is the number of ways an ordered set of size r can be given by a set of size n. The value of a permutation is given by nPr = ((n)! / (n - r)!), where n >= r >= 0. For permutations, set {A, B, C} is different than {B, C, A}.


Combinations:

A combination is the number of ways an unordered set of size r can be given by a set of size n. The value of a combination is given by nCr = ((n)! / ((n - r)! * r!)). For combinations, set {A, B, C} is the same as {B, C, A}.
